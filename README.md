# Restaurant-Page
## About the Project

This project was made as part of **The Odin Project** <a href="https://www.theodinproject.com/lessons/node-path-javascript-restaurant-page#project-solution">curriculum</a>.

This project is a cute restaurant page made for pandas. I built it because I like them and I thought it was a funny way to learn how to work with **npm** and **Webpack**.

Throughout this project, I learned how to deploy a bundled page in **GitHub Pages** and how to use `import`/ `export` and how to create modules to keep the code clean and organized. 

## Tech used
- HTML5, CSS3, JavaScript
- npm, Webpack

## How To Run Locally
1. Clone the repository: `git clone https://github.com/AlinC10/Restaurant-Page.git`
2. Open the terminal
3. Go to the project directory
4. Run `npm install` to install all the dependencies
5. Run `npx webpack` and `npx webpack serve` to launch a local server. Go to `http://localhost:8080/` in your favorite browser.
